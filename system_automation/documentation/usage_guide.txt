Linux System Automation - Usage Guide

Running Scripts:
----------------
- Run disk monitoring:   ./disk_monitor.sh
- Run user management:   ./user_management.sh add username
- Run backup:            ./backup.sh /path/to/dir
- Run process monitor:   ./process_monitor.sh
- Run system health:     ./system_health.sh
- Run security audit:    ./security_audit.sh
- Run network monitor:   ./network_monitor.sh
- Run performance log:   ./performance_logger.sh
- Run master script:     ./master_logger.sh

Interpreting Outputs:
---------------------
- Disk Monitor: Warning message if >80% usage.
- Backup: Creates .tar.gz file in backup folder.
- Process Monitor: Automatically restarts apache2 if stopped.
- Security Audit: Logs failed SSH attempts in security_audit.log.

Examples:
---------
$ ./backup.sh /home/user/documents
Output: Backup created at /backups/documents_2025-09-24.tar.gz

$ ./system_health.sh
Output: CPU Load: 0.45, Memory Used: 1024MB
